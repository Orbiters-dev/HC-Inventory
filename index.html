Cross-Platform SKU Mapping

Shopify + Amazon 제품 매핑 — barcode 기반 크로스 플랫폼 연결

DONE
🔗
Mapping Engine
barcode → ASIN → Product Name
두 개의 마스터 파일에서 barcode(EAN/UPC)를 키로 크로스 매핑. 매핑 안 되는 제품은 FNSKU fallback 사용.

Master Files

Shopify ProductsShopify Product List.xlsx — 437 variants, barcode + title
Amazon ProductsAmazon Product List.xlsx — 100 ASINs, EAN/UPC + FNSKU

Name Priority

1stShopify Product List → Title (primary)
2ndAmazon Product List → Title (fallback)
3rdCSV description field (last resort)
🔎
Detection
Unmapped ASIN & Missing Shopify Alert
새 ASIN이 CSV에 등장하면 자동 감지 → 리포트에 경고 표시 + GitHub Issue 알림.

Alerts

Unmapped ASINAmazon Product List에 없는 새 ASIN → 수동 추가 필요
Missing ShopifyFLT 제품이 Shopify Product List에 없을 때
ExceptionAMAZON_ONLY_ASINS — 아마존 전용 바코드 제품 (B0DFW5SSLC)

Operational Guide

월별 운영 절차

1️⃣
Step 1 — Monthly
Download Inventory Files
매월 초 이전 달 데이터 수집. Playwright 스크립트 실행 또는 수동 다운로드.
Commandpy -X utf8 collectors/download_inventory_web.py --month 2026-02
NoteAmazon MFA 필요 → WhatsApp 인증 선택
2️⃣
Step 2 — Monthly
Generate Report
다운로드 완료 후 리포트 생성.
Commandpy -X utf8 agents/inventory_v2/inventory_report.py
Outputoutput/inventory_report.html
3️⃣
Step 3 — Anytime
Check In-Transit
실시간 입고중 재고 확인 (API 호출).
Commandpy -X utf8 collectors/intransit_collector.py
Option--seller grosmimi — 특정 셀러만 조회
4️⃣
Step 4 — On Alert
Handle Unmapped ASINs
리포트에서 Unmapped ASIN 경고가 뜨면 Amazon Product List.xlsx에 수동 추가.
